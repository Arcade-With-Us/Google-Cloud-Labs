## 💻 **Execute in Cloud Shell** 
```
curl -LO raw.githubusercontent.com/Arcade-With-Us/Google-Cloud-Labs/refs/heads/main/Fundamentals%20of%20Cloud%20Logging/GSP610.sh

```

### For the first task
1. In a new tab, launch the App Engine bookshelf application. You should see the link at the top right of the App Engine Dashboard.
2. If you see an Internal Server Error, this is because the Datastore Index is not yet ready. Wait a minute and reload your browser.

![Demo](https://cdn.qwiklabs.com/DC%2BZ8FTM4sTQagu%2FIlrYOBnd4d%2BK%2FGK5nWozfS4Kfo4%3D)

3. Refresh the browser and click Add book. Then fill out the form like the following:
    * **Title**: `Test Book`
    * **Author**: `Jane Doe`
    * **Date Published**: `1/2/2003`
    * **Description**: Log test.
4. Your form should resemble the following:

   ![Demo](https://cdn.qwiklabs.com/0U61r7tiq6QLQI%2FSbCyXwfBeYLBRLCLvlEzvotRe%2BO0%3D)
